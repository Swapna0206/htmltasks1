HTML

Hypertext Markup Language(HTML) is the standard markup language for documents designed to be displayed in a web browser.It defines the content and structure of web content. It is often assisted by technologies such as Cascanding Style Sheets (CSS) and scripting languages such as javaScript, a programming language.

How it's used for

HTML is the basic building block of the web, and is used to create structured document with text,images,and other content.
